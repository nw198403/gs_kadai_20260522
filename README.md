# ①課題名
Coloracle（カラクル）〜カラー診断アプリ〜

## ②課題内容（どんな作品か）
- 色彩心理学に基づいて「今日のご自愛の一言」と「ラッキーアイテム」を届ける癒やし系メンタルケアアプリ
- 飼い主の愛犬であるチワワがこのアプリの案内人

## ③アプリのデプロイURL
デプロイしている場合はURLを記入（任意）

## ④アプリのログイン用IDまたはPassword（ある場合）
- なし

## ⑤工夫した点・こだわった点
- デザインを綺麗に見せたいと思い、Canvaで背景のデザイン動画を作成＆利用しました
- チワワをTop画面と、結果画面に表示しました（結果画面のチワワはカラー付きで）
- フォントもCanvaを参考に雰囲気に合うものを選び、カードの色やボタンの色もこだわりました
- カードを捲る音をつけました（荒川さんが紹介されていた効果音のサイトを活用させていただきました）

## ⑥難しかった点・次回トライしたいこと（又は機能）
- Claudeにかなり助けていただきました
- 毎日繰り返してもらえるような仕掛けを考えたかったのですが、十分な時間がなく、断念しました
